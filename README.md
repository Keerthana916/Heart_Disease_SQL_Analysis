# Heart Disease SQL Data Analysis Project

## Tools Used
- MySQL

## Project Overview
Performed SQL-based analysis on heart disease patient data to identify demographic patterns, disease prevalence, and medical risk factors.

## SQL Concepts Used
- Aggregate Functions
- Group By
- Case Satements
- Subqueries
- Windows Functions
- SQL Views

## Key Analysis
- Disease prevalence analysis
- Age group analysis
- Cholesterol risk analysis
- Gender comparison
- Exercise-induced angina analysis
